# DuplicaRighe
Apps Scripts per Google Sheet per duplicare righe in batch

Aggiunge un menù al foglio di Google Sheets dentro cui si installa.

Lavora con 2 modalità:

- permette di duplicare una singola riga (chiede quante copie deve fare)
- in una colonna si aspetta di trovare un numero che indichi quante copie deve farne

Esempio:
Tizio  4
Caio  2
Sempronio  3

Sulla base della colonna B l'output generato sarà:
Tizio
Tizio
Tizio
Tizio
Tizio
Caio
Caio
Caio
Sempronio
Sempronio
Sempronio
Sempronio
